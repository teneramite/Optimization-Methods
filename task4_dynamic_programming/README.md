## Описание задачи

Задача оптимального управления инвестиционным портфелем на 3 этапа.

**Начальное состояние:**
- ЦБ1 = 100 д.е.
- ЦБ2 = 800 д.е.
- Депозиты = 400 д.е.
- Свободные средства = 600 д.е.

**Управление:** Покупка/продажа долями по 25% от начального объема

**Ограничения:** ЦБ1 ≥ 30, ЦБ2 ≥ 150, Депозиты ≥ 100

**Комиссии:** ЦБ1 = 4%, ЦБ2 = 7%, Депозиты = 5%

## Алгоритм (уравнение Беллмана)

```
V(s,t) = max_u E[reward(s,u,scenario) + V(s',t+1)]

где:
- s - состояние (ЦБ1, ЦБ2, Деп, Своб)
- t - этап
- u - управление
- E[] - математическое ожидание (критерий Байеса)
```
