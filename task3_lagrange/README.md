## Описание задачи

**Вариант 7:**
```
Минимизировать: f(x,y) = (x-2)² + (y+3)²

При ограничениях:
  g₁(x,y) = 3x + 2y - 4 = 0
  g₂(x,y) = x - y - 1 = 0
```

## Алгоритм

1. Составить функцию Лагранжа: L = f + λ₁g₁ + λ₂g₂
2. Найти частные производные: ∂L/∂x, ∂L/∂y, ∂L/∂λ₁, ∂L/∂λ₂
3. Решить систему уравнений (приравнять к нулю)
4. Проверить характер точки через критерий Сильвестра
